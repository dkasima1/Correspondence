You can run this experiment by opening up an IDE such as brackets that is capable of running javascript and a live server that appears
in the browser. Each of the four experiments test a participants visual correspondence. Each experiment has a data_analysis.py program that compiles all participant data into one document. Each experiment has a link to view the experiment online and with instructions on how to do it.

Colors - http://www.bk.perceptionresearch.org/FlombaumBannu/

Colors_Randomized - http://www.bk.perceptionresearch.org/Colors_Randomized/

Orientation_Correspondence - http://www.bk.perceptionresearch.org/Orientation_Correspondence/

Inside_Colors - http://www.bk.perceptionresearch.org/InsideColors/
